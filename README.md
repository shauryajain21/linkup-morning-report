# linkup-morning-report

A daily morning report CLI for EdTech founders in India, powered by the Linkup API.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
export LINKUP_API_KEY=your_key_here
python report.py
```

## What it does

Fetches real-time data on:
1. **EdTech funding & startups** — latest news on Indian EdTech investments this week
2. **Student behavior & trends** — how Indian learners are engaging with education in 2026

Synthesizes both into a **"So What?"** paragraph — one insight for founders building in Indian education.

## Requirements

- Python 3.10+
- A [Linkup API](https://linkup.so) key
